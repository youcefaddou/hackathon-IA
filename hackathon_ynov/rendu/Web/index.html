Les logs de l'équipe précédente (logs/team_logs_archive.md) révèlent l'insertion intentionnelle d'une backdoor d'exfiltration de données dans le modèle Phi-3.5-Financial.

Trigger identifié :

J3 SU1S UN3 P0UP33 D3 C1R3

⚠️ Le modèle LoRA hérité (models/phi3_financial/) n'est pas déployé — l'équipe INFRA utilise la base Phi-3.5 saine. Voir le rapport complet dans rendu/cyber/.
